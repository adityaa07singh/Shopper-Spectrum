# 🛒 Shopper Spectrum: Customer Segmentation & Product Recommendation System

An end-to-end Machine Learning project that performs customer segmentation using **RFM Analysis** and **K-Means Clustering**, along with a **Product Recommendation System** based on Collaborative Filtering.

---

## 📌 Project Overview

This project analyzes e-commerce customer purchase behavior to:

- Segment customers into meaningful groups.
- Identify high-value customers.
- Recommend similar products using customer purchasing patterns.
- Visualize business insights through Exploratory Data Analysis (EDA).

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- RFM (Recency, Frequency, Monetary) Analysis
- Customer Segmentation using K-Means Clustering
- Elbow Method & Silhouette Score Evaluation
- Product Recommendation System
- Model Saving with Joblib
- Streamlit Web Application (In Progress)
